Welcome to Immich Companion

Save photos and videos from any browser tab to your self-hosted Immich, search your library from the address bar, and turn the new tab into a memory feed.

  1. 1

    Open your Immich server

    You need a working Immich instance you can sign into. The companion never talks to anyone else.

  2. 2

    Create an API key

    In Immich:

    1. Click your profile picture (top right) → Account Settings
    2. Open the API Keys section
    3. Click + New API Key and name it Browser Extension
    4. Pick the permissions below, then Create
    5. Copy the key — Immich shows it only once
    Permissions to grant click the row to copy the scope name
    asset.upload save images and videos to your library
    asset.read show search results, thumbnails, and new-tab background
    asset.view open assets when clicked
    album.read list albums in the settings dropdowns
    albumAsset.create add saved items to your default album (optional)
    sharedLink.create "Save & share" — create a public link on the spot
    Select all simplest — full read/write to your account. The extension still only uses the scopes listed under "Minimal".
    Older Immich builds (≤ 1.106) had a single all-or-nothing key. If you don't see scopes, the key has full access by default.
