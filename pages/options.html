Generate a new key from your Immich session

If you're signed into Immich in any browser tab — OAuth, SSO, or password — the extension can mint a fresh scoped API key without asking for credentials and replace the one above. Useful when rotating the key, after a server reset, or if your current key was revoked.

Required API key permissions click a row to copy the scope name
asset.upload save images and videos to your library
asset.read search results, thumbnails, and video playback
asset.download download originals from the popup's quick actions
asset.view open assets when clicked (older Immich builds)
album.read list albums in the settings dropdowns
albumAsset.create add saved items to your default album (optional)
asset.update archive saved items so they don't clutter your timeline (optional)
sharedLink.create "Save & share" — create a public link on the spot
Select all simplest — full read/write to your account. The extension still only uses the scopes listed under "Minimal".
In Immich → Account Settings → API Keys → + New API Key. Older Immich builds (≤ 1.106) had a single all-or-nothing key — if you don't see scopes, the key has full access by default.

About

Companion extension for self-hosted Immich.

  • Save to Immich — right-click any image or video.
  • Popup search — toolbar button → smart search of your library.
  • New tab — random photo + "On this day".
  • Share toolbar — Slideshow + Download-all on share pages.

Reset
