# Web Store Assets

Generated by `make_assets.py` (requires Pillow). Run from this folder:

    python3 make_assets.py

## Files

| File                              | Required? | Use                                        |
| --------------------------------- | --------- | ------------------------------------------ |
| `promo-tile-440x280.png`          | yes       | Small promo tile (Web Store listing)       |
| `promo-marquee-1400x560.png`      | optional  | Marquee promo tile (homepage placement)    |
| `screenshot-template-1280x800.png`| —         | Empty template; replace with real screenshots |

## Screenshots to provide before publishing

Capture **at least one** of these at 1280 × 800 (Chrome's "Capture screenshot" with the device toolbar set to 1280×800 works well):

1. **Options page** — `chrome-extension://<id>/pages/options.html` with the New Tab section open.
2. **New tab** — your library running with a real photo background and the "On this day" strip populated.
3. **Popup with results** — search for something with at least 6 hits.
4. **Save toast** — right-click an image, screenshot the green "Saved to Immich" toast in action.

Save the resulting PNGs into this folder. The Web Store listing accepts up to 5.

## Listing fields

When uploading to the Chrome Web Store dashboard:

- **Single purpose**: "Companion for self-hosted Immich servers — save, search, and share photos and videos."
- **Justifications** for permissions: see `PRIVACY.md` in the repo root for the rationale paragraph.
- **Privacy policy URL**: serve `PRIVACY.md` from a public GitHub Pages or gist URL.
